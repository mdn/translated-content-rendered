WebRTC (Web Real-Time Communication) (communication en temps réel web) est une API appelée par les applications web de tchat vidéo, d'appels vocaux et de partage de fichiers P2P.

WebRTC est constitué principalement de ces éléments :

getUserMedia
accorde l'accès à la caméra et / ou au microphone d'un appareil, et peut brancher leurs signaux à une connexion RTC.
RTCPeerConnection
configure les appels vocaux ou tchat vidéo
RTCDataChannels
fournit une méthode de définition des itinéraires de données pair-à-pair entre les navigateurs
