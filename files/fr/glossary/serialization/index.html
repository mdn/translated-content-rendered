--- title: Sérialisation slug: Glossary/Serialization tags: - CSS - Encodage - Glossaire - JavaScript - Sérialisation translation_of: Glossary/Serialization original_slug: Glossaire/Sérialisation ---

Le processus par lequel un objet ou une structure de données est traduit dans un format approprié pour un transfert sur un réseau, ou un stockage (par exemple dans un tampon de tableau ou un format de fichier).

Dans JavaScript, par exemple, vous pouvez sérialiser un objet d'une chaîne de caractères JSON par appel de la fonction JSON.stringify().

Les valeurs de CSS sont sérialisées par appel de la fonction CSSStyleDeclaration.getPropertyValue().
