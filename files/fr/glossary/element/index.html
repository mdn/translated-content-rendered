Un élément est une partie d'une page web. En XML et HTML, un élément peut contenir une donnée, un morceau de texte ou une image, ou même parfois ne rien contenir du tout. Un élément est typiquement constitué d'une balise ouvrante ayant quelques attributs, du contenu textuel et d'une balise fermante.

Les éléments et les balises ne sont pas la même chose. Les balises commencent ou clôturent un élément dans le code source, alors que les éléments font partie du DOM, le document qui sert de modèle pour afficher la page dans le navigateur.
