--- title: Fonction anonyme auto-exécutante slug: Glossary/Self-Executing_Anonymous_Function tags: - Glossary translation_of: Glossary/Self-Executing_Anonymous_Function original_slug: Glossaire/fonction_anonyme_auto-executante ---

Une fonction JavaScript qui se lance dès qu'elle est définie. Aussi connu en tant que IIFE (Immediately Invoked Function Expression ou Expression-fonction immédiatement invoquée).

Voir IIFE pour plus d'informations.
