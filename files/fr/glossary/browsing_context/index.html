Un contexte de navigation est l'environnement dans lequel un navigateur affiche un document. Dans les navigateurs modernes, il s'agit généralement d'un onglet, mais il peut s'agir d'une fenêtre ou encore seulement des parties d'une page, comme une <frame> ou une <iframe>.

Chaque contexte de navigation possède une origine spécifique, l'origine du document actif, ainsi qu'un historique qui énumère dans l'ordre tous les documents affichés.

La communication entre les contextes de navigation est sévèrement restreinte. Entre des contextes de la même origine, il est possible d'ouvrir et utiliser un canal BroadcastChannel.
