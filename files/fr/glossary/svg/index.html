Scalable Vector Graphics (SVG) est un format d'image vectorielle 2D basé sur une syntaxe XML.

Le W3C a commencé à travailler sur SVG à la fin des années 90, mais SVG n'est devenu populaire qu'avec la sortie d'Internet Explorer 9, supportant le SVG. Tous les principaux navigateurs le prennent maintenant en charge.

Basé sur une syntaxe XML, SVG peut être stylé avec CSS et être rendu interactif grâce à JavaScript. HTML5 permet à présent d'intégrer directement des balises SVG au sein d'un document HTML.

SVG étant un format d'image vectorielle, les images SVG peuvent être redimensionnées à l'infini, ce qui les rend inestimables pour la conception web adaptative, car elles permettent de créer des éléments d'interface qui s'adaptent à toutes les tailles d'écran. SVG fournit également un ensemble d'outils, comme du clipping, des masques, des filtres et des animations.
