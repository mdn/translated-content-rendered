Interface utilisateur et suite applicative par défaut de la plate-forme Firefox OS.

Une fois Firefox OS démarré, tout ce qui est affiché à l'écran est produit par la couche Gaia (e.g., l'écran de verrouillage, l'écran d'accueil, les applications standards). Gaia est entièrement implémenté en HTML, CSS, et JavaScript ; ses uniques interfaces vers le système d'exploitation sous-jacent se font via des APIs Web, celles-ci étant implémentées par la couche Gecko. Des applications tierces peuvent être installées aux côtés de Gaia.
