Uniform Resource Locator (URL) est une chaîne de texte qui spécifie où une ressource (telle qu'une page web, une image ou une vidéo) peut être trouvée sur Internet.

Dans le contexte de HTTP, les URL sont appelées "adresse web" ou "lien". Votre browser affiche les URL dans sa barre d'adresse, par exmple: https://developer.mozilla.org. Certains navigateurs affichent uniquement la partie d'une URL après le "//", c'est-à-dire le Domain name.

Les URL peuvent également être utilisées pour le transfert de fichiers (FTP) , les e-mails (SMTP), et d'autres applications.
