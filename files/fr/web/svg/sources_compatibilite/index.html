Les sources suivantes sont utilisées pour les tableaux de compatibilités des éléments SVG et de leurs attributs :

  • https://developer.mozilla.org/En/SVG_in_Firefox associé à son historique de révision pour Firefox
  • http://www.webkit.org/projects/svg/status.xml associé à son archive enregistré pour Webkit, Safari et Chrome
  • http://www.opera.com/docs/specs/opera9/svg/ ainsi que les pages pour Opera >= 9, http://www.opera.com/docs/specs/opera8/ pour Opera 8
  • http://blogs.msdn.com/b/ie/archive/2010/03/18/svg-in-ie9-roadmap.aspx pour connaître l'état du support sur IE9
  • Le graphique de support SVG sur Codedread.com pour les vérifications basiques par rapport à la suite de test W3C
  • Wikipedia pour des conseils basiques, non normatif
  • Svground.fr/ tuto assez complet sur le SVG
