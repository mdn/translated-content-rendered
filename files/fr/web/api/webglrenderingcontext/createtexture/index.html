La méthode WebGLRenderingContext.createTexture() de l'API WebGL crée et initialise un objet WebGLTexture.

Syntaxe

WebGLTexture gl.createTexture();

Paramètres

Aucun.

Valeur retournée

Un objet WebGLTexture auquel des images peuvent être liées.

Exemples

Voir aussi le tutoriel WebGL sur l'Utilisation de textures en WebGL.

Création d'une texture

var canevas = document.getElementById('canevas');
var gl = canevas.getContext('webgl');
var texture = gl.createTexture();
