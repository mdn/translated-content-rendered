La méthode valueOf() renvoie la valeur primitive encapsulée dans un objet BigInt.

Le code source de cet exemple interactif est disponible dans un dépôt GitHub. Si vous souhaitez contribuer à ces exemples, n'hésitez pas à cloner https://github.com/mdn/interactive-examples et à envoyer une pull request !

Syntaxe

bigIntObj.valueOf()

Valeur de retour

Un grand entier (big int) représentant la valeur primitive de l'objet BigInt courant.

Exemples

Utiliser valueOf()

typeof Object(1n); // object
typeof Object(1n).valueOf(); // bigint
