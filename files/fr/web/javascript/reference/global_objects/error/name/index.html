La propriété name est une chaîne de caractères représentant le nom du type d'erreur. La valeur initiale est "Error".

Description

Par défaut, les instances d'Error reçoivent le nom "Error". La propriété name, associée à la propriété message, est utilisée par la méthode Error.prototype.toString() pour créer une représentation de l'erreur sous la forme d'une chaine de caractères.

Exemples

Déclencher une erreur personnalisée

var e = new Error("Donnée malformée"); // e.name est "Error"

e.name = "ParseError";
throw e;
// e.toString() renverra "ParseError: Donnée malformée"
