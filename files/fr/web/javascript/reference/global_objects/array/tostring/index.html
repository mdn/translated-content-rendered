La méthode toString() renvoie une chaine de caractères représentant le tableau spécifié et ses éléments.

Le code source de cet exemple interactif est disponible dans un dépôt GitHub. Si vous souhaitez contribuez à ces exemples, n'hésitez pas à cloner https://github.com/mdn/interactive-examples et à envoyer une pull request !

Syntaxe

arr.toString()

Valeur de retour

Une chaîne de caractères qui représente les éléments du tableau.

Description

L'objet Array redéfinit la méthode toString d'Object. Pour les objets Array, la méthode toString() concatène les éléments du tableau et renvoie une chaîne contenant chacun des éléments, séparés par des virgules.

JavaScript appelle la méthode toString() automatiquement lorsqu'un tableau doit être représenté par une valeur texte ou lorsqu'on fait référence à un tableau dans une concaténation de chaines de caractères.

Sémantique d'ECMAScript 5

À partir de JavaScript 1.8.5 (Firefox 4), et en cohérence avec la 5e édition d'ECMAScript, la méthode toString() est générique et peut être utilisé avec n'importe quel objet. Object.prototype.toString() sera appelée, et la valeur résultante sera renvoyée.
