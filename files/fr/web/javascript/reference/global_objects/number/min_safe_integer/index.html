La constante Number.MIN_SAFE_INTEGER représente le plus petit entier représentable correctement en JavaScript (-(253 -1)).

Note : Pour représenter des entiers inférieurs à cette valeur, on pourra utiliser le type BigInt.

Description

La constante MIN_SAFE_INTEGER vaut -9007199254740991. Cette valeur provient du fait qu'en JavaScript, les nombres sont représentés en format à double précision selon la norme IEEE 754 et on ne peut représenter correctement que les nombres compris entre -(253-1) et 253 -1. Voir Number.isSafeInteger() pour plus d'informations.

MIN_SAFE_INTEGER étant une méthode statique de Number, il faut utiliser Number.MIN_SAFE_INTEGER()et non pas la méthode d'un objet Number qui aurait été instancié.

Exemples

Number.MIN_SAFE_INTEGER // -9007199254740991
-Math.pow(2, 53) -1     // -9007199254740991
