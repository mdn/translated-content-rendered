La propriété Intl.Collator.prototype représente l'objet prototype du constructeur Intl.Collator.

Attributs de Intl.Collator.prototype
Écrivable Non
Énumérable Non
Configurable Non

Description

Voir la page Collator pour une description appliquée aux instances de Intl.Collator.

Les instances de Intl.Collator héritent de Intl.Collator.prototype. Les modifications apportées à l'objet prototype sont propagées sur toutes les instances de Intl.Collator via l'héritage (chaîne de prototypes).

Propriétés

Intl.Collator.protoype.constructor
Une référence vers Collator.

Méthodes

Intl.Collator.prototype.compare
Un accesseur qui renvoie une fonction comparant deux chaînes de caractères, basée sur l'ordre de tri de l'objet Intl.Collator.
Intl.Collator.prototype.resolvedOptions()
Renvoie un nouvel objet dont les propriétés correspondent aux options de collation et de locales calculées lors de l'initialisation de l'objet.
