<p>L&apos;élément HTML <strong><code>&lt;xmp&gt;</code></strong> (pour <em>example</em>) affiche le texte entre les balises d&apos;ouverture et de fermeture sans interpréter le HTML qu&apos;il contient et en utilisant une police à chasse fixe. La spécification HTML 2 recommande un affichage suffisamment large pour contenir 80 caractères par ligne.</p> <div class="note"><strong>Note : </strong>Ne pas utiliser cet élément. <ul> <li>Il a été déprécié depuis HTML3.2 et n&apos;a pas été implémenté d&apos;une manière cohérente. Il a été complètement retiré du langage dans HTML5.</li> <li>Utilisez l&apos;élément <a href="/fr/docs/Web/HTML/Element/pre"><code>&lt;pre&gt;</code></a> ou, si sémantiquement approprié, l&apos;élément <a href="/fr/docs/Web/HTML/Element/code"><code>&lt;code&gt;</code></a>. Notez qu&apos;il vous faudra échapper les caractères &apos;<code>&lt;</code>&apos; et &apos;<code>&gt;</code>&apos; pour qu&apos;ils ne soient pas interprétés.</li> <li>Une police à chasse fixe peut être obtenu sur n&apos;importe quel élément en utilisant des règles <a href="/fr/docs/CSS">CSS</a>, par la valeur <code>monospace</code> avec la propriété <a href="/fr/docs/Web/CSS/font-family"><code>font-family</code></a>.</li> </ul> </div> <h2 id="Attributs">Attributs</h2> <p>Cet élément n&apos;a aucun autre attribut en dehors <a href="/fr/docs/Web/HTML/Attributs_universels">des attributs universels</a>, communs à tous les éléments.</p> <h2 id="Interface_DOM">Interface DOM</h2> <p>Cet élément implémente l&apos;interface <a href="/fr/docs/Web/API/HTMLElement"><code>HTMLElement</code></a>.</p> <div class="note"> <p><strong>Note d&apos;implémentation : </strong>Jusqu&apos;à Gecko 1.9.2 inclus, Firefox implémentait l&apos;interface <a href="/fr/docs/Web/API/HTMLSpanElement"><code>HTMLSpanElement</code></a> pour cet élément.</p> </div> <h2 id="Compatibilité_des_navigateurs">Compatibilité des navigateurs</h2> <div class="hidden">Le tableau de compatibilité de cette page a été généré à partir de données structurées.
