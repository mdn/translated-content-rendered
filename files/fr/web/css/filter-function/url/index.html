La fonction url() permet d'utiliser un filtre SVG afin de modifier l'apparence d'une image.

Syntaxe

url(emplacement)

Paramètres

emplacement
L'URL (<url>) d'un fichier XML qui définit un filtre SVG. On peut ajouter une ancre sur cette URL afin d'indiquer un filtre spécifique.

Exemple

url(ressources.svg#c1)
