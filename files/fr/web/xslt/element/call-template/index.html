Référence XSLT/XPath : Éléments XSLT, Fonctions EXSLT, Fonctions XPath, Axes XPath

L'élément <xsl:call-template> invoque un modèle nommé.

Syntaxe 

<xsl:call-template name=NOM>
	<xsl:with-param> [optionnel]
</xsl:call-template>

Attribut obligatoire

name
Définit le nom du modèle à invoquer.

Attributs optionnels

Aucun.

Type

Instruction, apparaît dans un modèle.

Définition

XSLT 1.0, section 6.

Support Gecko

Supporté.
