WebGL (Web Graphics Library) est une API JavaScript pour produire des graphismes 2D et 3D interactifs.

Le Groupe Khronos maintient WebGL qui est basé sur OpenGL ES 2.0.

Vous pouvez faire appel à WebGL à partir de l'élément HTML <canvas> qui fournit une surface de rendu.

Tous les principaux navigateurs prennent maintenant en charge WebGL, mais sa disponibilité dépend également de facteurs externes (e.g. support par le GPU).
