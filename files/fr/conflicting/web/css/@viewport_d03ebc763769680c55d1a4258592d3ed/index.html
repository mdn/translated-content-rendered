Le descripteur max-zoom, rattaché à la règle @ @viewport, permet de définir le niveau de zoom maximal au-delà duquel le navigateur n'augmentera pas le niveau de zoom (que ce soit une action automatique ou une action de l'utilisateur).

Un facteur de zoom égal à 1.0 ou à 100% correspond à l'absence de zoom. Si on utilise des valeurs supérieures, cela correspondra à un niveau de zoom plus élevé. À l'inverse des valeurs inférieures traduiront un « dézoom ».

Syntaxe

/* Valeur avec un mot-clé */
max-zoom: auto;

/* Nombres : type <number> */
max-zoom: 0.8;
max-zoom: 2.0;

/* Valeurs proportionnelles : type <percentage> */
max-zoom: 150%;

Valeurs

auto
L'agent utilisateur détermine la limite du niveau de zoom applicable pour le document.
<number>
Un nombre positif qui correspond à la limite haute du niveau de zoom (cf. <number>).
<percentage>
Un pourcentage positif qui correspond à la limite haute du niveau de zoom (cf. <percentage>).

Syntaxe formelle

auto | <number> | <percentage>
