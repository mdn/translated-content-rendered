La propriété WeakSet.prototype représente le prototype du constructeur WeakSet.

Attributs de WeakSet.prototype
Écrivable Non
Énumérable Non
Configurable Non

Description

Toutes les instances de WeakSet héritent de WeakSet.prototype. Le prototype du constructeur peut être utilisé pour ajouter des méthodes et/ou des propriétés à toutes les instances de WeakSet.

WeakSet.prototype est un objet ordinaire :

Object.prototype.toString.call(WeakSet.prototype); // "[object Object]"

Propriétés

WeakSet.prototype.constructor
Cette propriété renvoie la fonction qui a créé le prototype de l'instance. Par défaut, ce sera la fonction native WeakSet.

Méthodes

WeakSet.prototype.add(valeur)
Cette méthode permet d'ajouter une nouvel objet avec une valeur donnée à l'objet WeakSet.
WeakSet.prototype.delete(valeur)
Cette méthode retire l'élément associé à valeur. WeakSet.prototype.has(valeur) renverra false une fois l'opération effectuée.
WeakSet.prototype.has(valeur)
Cette méthode renvoie un booléen indiquant si oui ou non un élément est présent avec cette valeur au sein de l'objet WeakSet.
WeakSet.prototype.clear()
Cette méthode retire tous les éléments de l'ensemble WeakSet.
