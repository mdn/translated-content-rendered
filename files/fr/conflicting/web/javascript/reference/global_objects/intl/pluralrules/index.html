La propriété Intl.PluralRules.prototype représente le prototype du constructeur Intl.PluralRules.

Attributs de Intl.PluralRules.prototype
Écrivable Non
Énumérable Non
Configurable Non

Description

Voir PluralRules pour la description des instances Intl.PluralRules.

Les instances de Intl.PluralRules héritent de Intl.PluralRules.prototype. Les modifications apportées au prototype seront héritées par l'ensemble des instances de Intl.PluralRules.

Propriétés

Intl.PluralRules.prototype.constructor
Une référence à Intl.PluralRules.

Méthodes

Intl.PluralRules.prototype.resolvedOptions()
Cette méthode renvoie un nouvelle objet dont les propriétés reflètent la locale et les options de collations calculées lors de l'initialisation de l'objet.
Intl.PluralRules.prototype.select()
Cette méthode renvoie une chaîne de caractères (String) qui indique quelle forme de règle de nombre est utilisée pour le formatage.

Spécifications
