HTML törés elem felület

A DOM törés elemek a HTMLBRElement (vagy HTML 4 HTMLBRElement) felületet fejtik ki, amelyek a HTMLElementből öröklődnek, de nem határoz meg további tagokat a HTML5 verzióban. A bevezetett további tulajdonság szintén elavult a HTML 4.01 verzióban.
