A Cascading Style Sheets (CSS) egy stíluslap nyelv, mellyel a HTML-ben vagy XML-ben (és más XML nyelveken, mint az XHTML vagy a SVG) írt dokumentumok megjelenítését lehet leírni.

A CSS a nyílt web egyik fő nyelve, amely a W3C specifikációnak megfelelően szabványosítva van a webböngészőkben. Szintenként van fejlesztve, a CSS1 ma már elavult, a CSS2.1 használata az ajánlott, a CSS3 pedig már kisebb modulokra lett osztva, jelenleg is a szabványosítása zajlik.

  • CSS Bevezetés

    Ha új vagy a webfejlesztésben, olvasd el a CSS alapismeretek útmutatót, hogy megtudd mi az a CSS és hogyan használd.

  • CSS Oktatóanyagok

    CSS tanulási területünk rengetek oktatóanyagot tartalmaz, amelyek a kezdő szinttől a szakértelemhez vezetnek.

  • CSS ReferenciákA teljeskörű CSS referencia a tapasztalt webes fejlesztők számára a CSS minden tulajdonságát és fogalmát írja le.

Eszközök a CSS fejlesztéshez

  • A W3C CSS Validation Service segítségével ellenőrizheted, hogy a CSS kódod érvényes-e. Ez felbecsülhetetlen hibakereső eszköz.
  • A Firefox Developer Tools lehetőséget nyújt megtekinteni és szerkeszteni az oldal élő CSS-ét az Inspector és a Style Editor eszközök segítségével.
  • A Web Developer bővítmény lehetővé teszi az élő CSS követését és szerkesztését a megfigyelt webhelyeken.
