Metoda clear() usuwa wszystkie elementy obiektu Set.

Składnia

mySet.clear();

Zwracana wartość

undefined.

Przykłady

Użycie metody clear

var mySet = new Set();
mySet.add(1);
mySet.add('foo');

mySet.size;       // 2
mySet.has('foo'); // true

mySet.clear();

mySet.size;       // 0
mySet.has('bar')  // false
