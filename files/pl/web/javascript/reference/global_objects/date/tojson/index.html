Składnia

dateObj.toJSON()

Opis

toJSON() zwraca ciąg znaków (using toISOString()) reprezentujący wartość obiektu Date.

Przykłady

Użycie toJSON()

var jsonDate = (new Date()).toJSON();
var backToDate = new Date(jsonDate);

console.log(jsonDate); //2015-10-26T07:46:36.611Z
