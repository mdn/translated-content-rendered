Podsumowanie

Metoda deleteRule usuwa regułę stylu bieżącego obiektu arkusza stylów.

Składnia

stylesheet.deleteRule(index)

Parametry

  • index jest długą liczbą reprezentującą pozycję reguły.

Przykład

 myStyles.deleteRule(0);
