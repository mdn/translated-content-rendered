Funkcja function-available określa czy podana funkcja jest dostępna i zwraca wartość logiczną prawda lub fałsz.

Składnia 

function-available( nazwa )

Argumenty

name
Nazwa funkcji do ocenienia.

Zwraca

Wartość logiczną true lub false.

Definicja

XPath 1.0 15

Gecko

Obsługuje.
