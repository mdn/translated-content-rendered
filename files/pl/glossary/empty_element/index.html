Pusty element to element HTML, SVG lub MathML który nie może mieć żadnych węzłów dziecięcych (tj. elementów zagnieżdżonych lub węzłów tekstowych).

Specyfikacje HTML, SVG i MathML określają bardzo dokładnie, co każdy element może zawierać. Wiele kombinacji nie ma znaczenia semantycznego, na przykład element <audio> zagnieżdżony wewnątrz elementu <hr>.

W HTML, użycie znacznika zamykającego na pustym elemencie jest zazwyczaj nieważne. Na przykład, <input type="text"></input> jest nieprawidłowy.

Puste elementy w HTML są następujące:

  • <area>
  • <base>
  • <br>
  • <col>
  • <embed>
  • <hr>
  • <img>
  • <input>
  • <keygen>(HTML 5.2 Projekt usunięty)
  • <link>
  • <meta>
  • <param>
  • <source>
  • <track>
  • <wbr>
