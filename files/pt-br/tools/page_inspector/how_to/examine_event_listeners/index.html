No Firefox 33 você verá um ícone "ev" no HTML Pane, próximo a elementos que tem eventos de escuta como mostra a figura abaixo:

Clicando no ícone, então você verá um popup listando todos eventos de escuta para este elemento:

Cada linha contém:

  • um botão pause: clique neste para ir ao evento de escuta no Debugger, onde você poderá setar um breakpoint in it
  • o nome do evento
  • o nome, o número da linha para o evento de escuta: clique neste para ver a função de escuta no popup
  • o label indicando se o evento bubbles
  • a label indicando o sistema que define o evento. O Firefox pode mostrar:
    • evento padrão - DOM
    • eventos jQuery
    • eventos React (novo no Firefox 53)
