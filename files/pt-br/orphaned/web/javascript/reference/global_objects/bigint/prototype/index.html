A propriedade BigInt.prototype representa o protótipo para o  construtor BigInt .

Property attributes of BigInt.prototype
Writable no
Enumerable no
Configurable no

Descrição

Todas instância de BigInt herdam de BigInt.prototype. O objeto protótipo do construtor BigInt pode ser modificado para afetar todas instâncias de BigInt .

Propriedades

BigInt.prototype.constructor
Retorna a função que cria instâncias deste objeto. Por padrão é o objeto
BigInt.

Métodos

BigInt.prototype.toLocaleString()
Retorna uma string com uma representação sensível ao idioma para este número. Sobrescreve o método Object.prototype.toLocaleString()
 
BigInt.prototype.toString()
Retorna uma string respresentando o objeto específicado em um base específica. Sobrescreve o método Object.prototype.toString() .
BigInt.prototype.valueOf()
Retorna o valor primitivo de um objeto específicado. Sobrescreve o método Object.prototype.valueOf().
