O Valor inicial de uma propriedade CSS é o seu valor padrão, como listado em sua tabela de definição. O uso do valor inicial varia caso a propriedade seja herdada ou não.

  • Para propriedades herdadas, o valor inicial é usado apenas em seu elemento raíz, desde que nenhum valor especificado seja fornecido.
  • Para propriedades não-herdadas, o valor inicial é usado em todos os elementos, enquanto nenhum valor especificado é fornecido

Nota: Você pode especificar explicitamente um valor inicial, utilizando a palavra-chave initial
