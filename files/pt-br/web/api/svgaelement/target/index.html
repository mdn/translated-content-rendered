O SVGAElement.target propriedade somente ler de SVGAElement retorna um objeto SVGAnimatedString que especifica a porção de um alvo sendo ele "window", "frame" ou "pane" no qual um documento será aberto quando o link for acionado.

Esta propriedade é usada quando existem dois ou mais possiveis alvos(destinos) para o documento, por exemplo, quando o documento pai é um arquivo .html ou .xhtml com varias telas (multi-frame).

Sintaxe

myLink.target = 'value';

Valor

Um SVGAnimatedString indica o destino final do recurso que abre o documento assim que o link é acionado.

Exemplo

linkRef.target ='_blank';
