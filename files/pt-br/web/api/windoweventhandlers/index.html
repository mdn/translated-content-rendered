WindowEventHandlers descreve os manipuladores de eventos comuns a várias interfaces como Window, ou HTMLBodyElement e  HTMLFrameSetElement. Cada uma dessas interfaces podendo implementar manipuladores de eventos específicos adicionais.

WindowEventHandlers não é uma interface e nenhum objeto desse tipo pode ser criado.

Properties

As propriedades de evento, no formulário onXYZ, são definidas no WindowEventHandlers, e implementadas por Window, e WorkerGlobalScope para desenvolvedores web.

WindowEventHandlers.onafterprint
É uma EventHandler representando o código que será chamando quando o evento afterprint é invocado.
WindowEventHandlers.onbeforeprint
É um EventHandler representando o código que será chamado quando o evento beforeprint é invocado.
WindowEventHandlers.onbeforeunload
É um EventHandler representando o código que será chamado quando o evento  beforeunload é invocado.
WindowEventHandlers.onhashchange
É um EventHandler representando o código que será chamado quando o evento hashchange é invocado.
WindowEventHandlers.onlanguagechange
É um EventHandler representando o código que será chamado quando o evento languagechange é invocado.
WindowEventHandlers.onmessage
É um EventHandler representando o código que será chamado quando o evento  message é invocado.
WindowEventHandlers.onoffline
É um EventHandler representando o código que será chamado quando o evento  offline é invocado.
WindowEventHandlers.ononline
É um EventHandler representando o código que será chamado quando o evento  online é invocado.
WindowEventHandlers.onpagehide
É um EventHandler representando o código que será chamado quando o evento pagehide é invocado.
WindowEventHandlers.onpageshow
É um EventHandler representando o código que será chamado quando o evento pageshow é invocado.
WindowEventHandlers.onpopstate
É um EventHandler representando o código que será chamado quando o evento popstate é invocado.
WindowEventHandlers.onresize
É um EventHandler representando o código que será chamado quando o evento resize é invocado.
WindowEventHandlers.onstorage
É um EventHandler representando o código que será chamado quando o evento storage é invocado.
WindowEventHandlers.onunload
É um EventHandler representando o código que será chamado quando o evento unload é invocado.

Métodos

Esta interface não define métodos.
