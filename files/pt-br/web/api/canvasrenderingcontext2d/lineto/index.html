O método CanvasRenderingContext2D.lineTo() da API Canvas 2D conecta o último ponto do sub-caminho (sub-path) para as coordenadas x, y, através de uma linha (mas na realidade não a desenha).

Sintaxe

void ctx.lineTo(x, y);

Parâmetros

x
O valor da coordenada x que indica o fim da linha.
y
O valor da coordenada y que indica o fim da linha.

Exemplos

Usando o método lineTo

sto é só um simples trecho de código que usa o método lineTo. Use o beginPath() para indicar o path onde será desenhada a linha, mova a caneta (pen) com moveTo() e use o método stroke() para desenhar a linha.

HTML

<canvas id="canvas"></canvas>

JavaScript

var canvas = document.getElementById('canvas');
var ctx = canvas.getContext('2d');

ctx.beginPath();
ctx.moveTo(50, 50);
ctx.lineTo(100, 100);
ctx.stroke();
