Especifica um event listener para receber eventos dischargingtimechange. Esses eventos ocorrem quando a propriedade dischargingTime (tempo de descarregamento) da bateria é atualizada.

Sintaxe

navigator.battery.ondischargingtimechange = funcRef

Onde funcRef é uma função para ser chamada quando o evento dischargingtimechange ocorre.
