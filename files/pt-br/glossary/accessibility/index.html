Acessibilidade na Web (Web Accessibility) (A11Y) refere-se às melhores práticas para manter um site útil, apesar das restrições físicas e técnicas. A acessibilidade da Web é formalmente definida e discutida no W3C através de Web Accessibility Initiative (WAI).
