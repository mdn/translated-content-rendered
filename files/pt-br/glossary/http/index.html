HTTP (HyperText Transfer Protocol) é o protocolo básico que habilita a transferência de arquivos na Web, normalmente entre um navegador e um servidor para que os humanos possam lê-los. A versão atual da especificação HTTP é chamada HTTP / 2.

Como parte de um URI, o "http: //" é chamado de "esquema" e geralmente fica no início de um endereço, por exemplo em "https://developer.mozilla.org" para indicar ao navegador que solicite o documento usando o protocolo HTTP. O https neste caso refere-se à versão segura do protocolo HTTP, SSL (também chamado TLS).

O HTTP é textual (toda a comunicação é feita em texto simples) e sem estado (nenhuma comunicação está ciente das comunicações anteriores). Esta propriedade torna ideal para humanos ler documentos (sites) na web. No entanto, o HTTP também pode ser usado como base para os serviços da Web REST de servidor para servidor ou solicitações AJAX dentro de sites para torná-los mais dinâmicos.
