Um nome de domínio é um endereço de uma página na Internet. Nomes de domínios são usados em URLs para identificar a qual servidor uma página específica pertence. O domínio consiste de uma sequência hierárquica de nomes (rótulo) separados por períodos (pontos) e terminando com uma extensão.
