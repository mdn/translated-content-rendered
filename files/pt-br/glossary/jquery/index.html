jQuery é uma Biblioteca JavaScript que visa a simplificação na manipulação do DOM , nas chamadas de AJAX e no gerenciamento de Eventos . É muito utilizado por desenvolvedores de Javascript.

jQuery usa o formato $(seletor).acão() para atribuir um evento a um elemento. Resumindo, $(seletor) chamará jQuery, que selecionará elemento(s) com base no 'seletor' e atribuirá um evento API chamado .acão().

$(document).ready(function(){
  alert("Hello World!");
  $("#blackBox").hide();
});

O codigo acima tem a mesma funcionalidade que o codigo abaixo:

window.onload = function() {
  alert( "Hello World!" );
  document.getElementById("blackBox").style.display = "none";
};
