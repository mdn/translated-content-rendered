flex é um novo valor adicionado à propriedade display. Junto com o inline-flex, este valor faz com que o elemento aplicado seja um flex container, e seus filhos se transformem então em flex item.Os itens então farão parte de um layout flex, e todas as propriedades definidas no CSS Flexbox poderão ser aplicadas.

A propriedade flex é um atalho para as propriedades flexboxflex-grow, flex-shrink e flex-basis.

Além disso, <flex> pode ser também uma unidade flexível em um CSS Grid Layout.
