IndexedDB é uma API de web para armazenar volumosas estruturas de dados dentro dos navegadores e indexá-los para buscas de alta performance. Sendo um RDBMS baseado em SQL, IndexedDB é um sistema de banco de dados transacionais. Porém ele usa objetos JavaScript ao invés de tabelas em colunas fixas para armazenar os dados.
