RandomSource representa uma fonte criptografada segura de números aleatórios. É disponível via objeto Crypto do objeto global: Window.crypto em páginas Web, WorkerGlobalScope.crypto em trabalhadores.

RandomSource não é uma interface e nenhum objeto deste tipo pode ser criado.

Propriedades

RandomSource não define ou herda nenhuma propriedade.

Métodos

RandomSource.getRandomValues()
Completa o ArrayBufferView com valores aleatoriamente criptografados.
