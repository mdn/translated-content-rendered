O CryptoKey.type é uma propriedade de apenas leitura que indica o tipo de key: se for uma key para um algoritmo simétrico ("secret") ou, para um algoritmo assimétrico, ("public" ou "private", dependendo do seu propósito).

Sintaxe

result = key.type

Valor de retorno

  • result é um valor enumerado que pode ser:
    • "secret" representa uma key única utilizada para criptografar ou descriptografar uma mensagem que faz uso de um algoritmo simétrico.
    • "public" representa uma key pública e compartilhável usada em um algoritmo assimétrico.
    • "private" representa uma key utilizada em um algoritmo assimétrico que deve ser mantida em privado.
