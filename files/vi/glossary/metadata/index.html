Metadata (Siêu dữ liệu) là — in its very simplest definition — dữ liệu mô tả dữ liệu. Chẳng hạn, tài liệu HTML là dữ liệu, nhưng HTML có thể chứa metadata trong phần tử <head> để mô tả tài liệu đó — như là ai viết nó, và tóm lược trang web.
