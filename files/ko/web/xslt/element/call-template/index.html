XSLT/XPath 참고 문서: XSLT 요소, XPath 함수, XPath 축

<xsl:call-template> 요소는 이름 붙인 템플릿을 호출합니다.

문법

<xsl:call-template name=NAME>
	<xsl:with-param> [optional]
</xsl:call-template>

필수 속성

name
호출하고 싶은 템플릿의 이름을 지정합니다.

선택 속성

없음.

타입

명령, 템플릿 안에 나타남.

정의

XSLT section 6, Named Templates

지원함.
