スラグは、通常はURLの最後にあるWebアドレスの固有の識別部分です。
MDNのコンテキストでは、"<locale> / docs /" に続くURLの部分です。

親文書の下に新しい文書が作成されたときの最終的な構成要素でもあります。
たとえば、このページのスラグは Glossary/スラグ です。
