  1. WindowTimers

概要

setInterval を使用して設定された繰り返し動作をキャンセルします。

構文

window.clearInterval(intervalID)
  • intervalID: キャンセルする繰り返し動作の識別子。この ID は setInterval() の戻り値です。

例

setInterval().example の例を参照して下さい。
