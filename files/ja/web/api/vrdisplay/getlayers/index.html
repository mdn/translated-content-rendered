VRDisplay インターフェイスの getLayers() メソッドは，VRDisplay で現在表示されているレイヤを返します．

シンタックス

var myLayers = vrDisplayInstance.getLayers();

パラメータ

なし．

戻り値

VRLayer オブジェクトの配列．
