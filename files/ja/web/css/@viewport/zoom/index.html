zoom は CSS の記述子で、 @viewport アット規則で定義された文書の表示倍率の初期値を設定します。

表示倍率 が 1.0 または 100% が、拡大縮小なしに対応します。より大きい値は拡大、より小さい値は縮小です。

構文

/* キーワード値 */
zoom: auto;

/* <number> 値 */
zoom: 0.8;
zoom: 2.0;

/* <percentage> 値 */
zoom: 150%;

値

auto
ユーザーエージェントが文書の表示倍率の初期値を設定します。ユーザーエージェントは表示倍率を決めるために、文書が表示されるキャンバス領域の寸法を使用することがあります。
<number>
表示倍率として使われる非負の数値です。
<percentage>
表示倍率として使われる非負のパーセント値です。

公式定義

関連する @規則@viewport
初期値auto
パーセンテージthe zoom factor itself
計算値auto, or a non-negative number or percentage as specified

形式文法

auto | <number> | <percentage>

例

ビューポートの zoom 係数の設定

@viewport {
  zoom: 2.0;
}
