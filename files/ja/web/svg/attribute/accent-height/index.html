accent-height属性は原点からアクセント文字の上端までの距離を定義します。これはフォント座標系で測られる距離です。

1つの要素のみがこの属性を使用します: <font-face>

font-face

<font-face>に対して、accent-heightは原点からアクセント文字の上端までの距離を定義します。これはフォント座標系で測られる距離です。
