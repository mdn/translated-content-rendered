min-zoom は CSS の記述子で、 @viewport アット規則で定義され、文書の最大表示倍率を設定します。ブラウザーは自動的にもユーザーのリクエストでも、これより拡大することができません。

表示倍率 が 1.0 または 100% が、拡大縮小なしに対応します。より大きい値は拡大、より小さい値は縮小です。

構文

/* キーワード値 */
max-zoom: auto;

/* <number> 値 */
max-zoom: 0.8;
max-zoom: 2.0;

/* <percentage> 値 */
max-zoom: 150%;

値

auto
ユーザーエージェントが文書の表示倍率の上限を設定します。
<number>
非負の数値で、表示倍率の上限です。
<percentage>
非負のパーセント値で、表示倍率の上限です。

公式定義

関連する @規則@viewport
初期値auto
パーセンテージthe zoom factor itself
計算値auto, or a non-negative number or percentage as specified

形式文法

auto | <number> | <percentage>

例

max-zoom の設定

@viewport {
  max-zoom: 1.5;
}
