  1. Slotable

Slotable mixin は、ノードを <slot> 要素のコンテンツにする機能を定義します。— 次の機能が Element と Text の両方に含まれています。

プロパティ

Slotable.assignedSlot 読取専用
ノードが挿入されている <slot> を返します。

メソッド

なし。
