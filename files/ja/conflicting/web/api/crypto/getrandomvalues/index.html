RandomSource は、暗号的に安全な乱数値のソースを表します。これは、グローバルオブジェクトの Crypto オブジェクトを通して利用可能です。ウェブページ上では Window.crypto、Worker 内では WorkerGlobalScope.crypto が利用できます。

RandomSource は、インターフェイスでも、作成できるこの種類のオブジェクトでもありません。

プロパティ

RandomSource はどのプロパティも定義または継承しません。

メソッド

RandomSource.getRandomValues()
渡された ArrayBufferView を意味不明の乱数値で埋めます。
