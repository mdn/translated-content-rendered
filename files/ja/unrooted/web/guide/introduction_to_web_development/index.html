Web開発の初歩

ここに示すリンクは、Web開発を始めたばかりの方であってもWebのすごさへの新しい領域に踏み込もうとしている方であっても手助けになるはずです。

注: 本ページの推奨リソースは変更される可能性があります。

 

項目

 

HTML

  • Hypertext Mark-up Langage (HTML)の基本 - HTMLとはいったい何か?
  • webページの基本構造 - doctype とドキュメント'ツリー'
  • 基本的なHTML要素  - カテゴリ別に説明された、構造、ヘッダー、リスト、フォーム要素ほか
  • HTML初心者へのチュートリアル - 上述の基礎を含めたチュートリアルと練習
  • MDC HTML要素リファレンスガイド - 詳細を含めGecko/FirefoxがサポートするHTML要素の包括的ガイド
はじめに 

CSS

  • 初心者のためのCSSのMDCガイド - 基本的な概念と基礎をカバーするCSSのまさに初心者用ガイド
  • MDC CSSリファレンスガイド - Gecko/Firefoxのレベルで詳細に記載したCSSの完全ガイド
  • CSSでスタイリングするためのW3Cの紹介 - 初心者がwebページをスタイリングするための分かりやすいガイド
  • よくあるCSSの質問 - 初心者によくあるQ&A
  • 中級レベルのCSSの概念 - グループ分け、pseudo-classesほか
はじめに 
中級者向け

JavaScript

  • 初心者向けMDC Javascriptガイド - Javascriptとは何か? どうやって役に立つのか?
  • MDC JavaScriptリファレンスガイド - 定期的にアップデートされる、初心者から上級者まですべてのレベルのための包括的ガイド
  • Crockford on JavaScript - Javascript言語に関し徹底的に追求したビデオシリーズ
  • Javascriptの再入門 - 中級者レベルの開発者向けJavascriptプログラミング言語の要約
  • 雄弁なJavaScript - 中級から上級向けJavascript技術の包括ガイド
  • 本質的なJavaScript設計パターン - 本質的なJavascriptの設計パターンの紹介
  • オブジェクト指向JavascriptのためのMDCの紹介 - Javascriptオブジェクトモデルについて学習する
  • Christian HeilmannのJavaScript最適学習 - Javascriptを書くときお決まりの(そんなにお決まりでないものも)最適な練習について学習する
  • 上級者向けJavaScriptを学ぶ - John Resigによる上級者向けJavaScriptガイド
  • Crockford on Advanced JavaScript - 上級者向けJavascript概念に関する3部構成のビデオシリーズ

リソース

Opera's Web Standards Curriculum
webデザイン、HTML、HTML5、CSS、Javascriptとそのアクセス法について。多くの分野を通じてweb開発の基礎を学習しようとしている初心者にとってここはよいスタートポイントになります。
SitePoint
異なるブラウザーやブラウザーに関する既知のバグを通じてのサポートにも言及している、HTML、CSS、Javascriptを学習するために信頼できるサイト
HTMLDog
初心者向け、HTMLとCSSに関する優秀で包括的なリファレンスサイト
Google's HTML, CSS, and Javascript from the Ground Up
Googleのプロのweb開発者による易しいビデオチュートリアルがHTML、CSS、Javascriptの基本をカバーします
CSSTutorial.net Beginner Tutorials
A broad range of useful text and video tutorials that cover the basics through to intermediate aspects of CSS.
Tizag CSS Tutorials
An easy-to-follow reference for those wishing to learn CSS through short, concise tutorials.
jQuery Fundamentals
An open-source reference book featuring detailed introductions to both JavaScript and jQuery Fundamentals for beginners.
JavaScript From Null: A Video Series
A video series on JavaScript for absolute beginners looking for more 'visual'-based learning
