La proprieta' in sola lettura dir dell'interfaccia Notification indica la direzione del testo della notifica, puo' essere specificata nell'opzione dir del costruttore di Notification() .

Note: This feature is available in Web Workers.

Sintassi

var direzione = Notification.dir;

Valori

La DOMString specifica la direzione el testo. I possibili valori sono:

  • auto: adotta  il comportamento delle impostazioni del browser (valore di  default.)
  • ltr: da sinistra a destra. ( es. |testo notifica                | )
  • rtl : da destra a sinistra.( es. |                 testo notifica|) 

Note: La maggior parte dei browser ignora le opzioni esplicite settate e funzionano con le proprie impostazioni.

Esempi

Il seguente frammento di codice crea un oggetto opzioni, da passare al costruttore Notification().

var opzioni = {
  body: 'Amo JavaScript!',
  dir: 'rtl'
}

var n = new Notification('Mio titolo',opzioni);

n.dir // dovrebbe ritornare 'rtl'
