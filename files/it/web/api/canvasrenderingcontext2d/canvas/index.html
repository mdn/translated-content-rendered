La proprietà CanvasRenderingContext2D.canvas è un riferimento di sola lettura verso il canvas HTMLCanvasElement che contiene il context corrente. Può essere null se non vi è un elemento <canvas> associato.

Sintassi

ctx.canvas;

Esempi

Dato il seguente elmento  <canvas>:

<canvas id="canvas"></canvas>

È possibile ottenere un riferimento all'elemento canvas da cui è ottenuto questo CanvasRenderingContext2D utilizzando la property canvas:

var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");
ctx.canvas // HTMLCanvasElement
