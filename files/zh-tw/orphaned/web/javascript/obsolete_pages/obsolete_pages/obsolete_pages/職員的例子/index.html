職員的例子

本章的剩餘部分會使用如下所示的職員的階層圖。

圖 8.1: 簡單的物件的階層

本範例使用了下面的物件︰

  • Employee 具有 name（其值預設為空字串）以及 dept（其值預設為 "general"）屬性。
  • Manager 以 Employee 為基礎。他加入了 reports 屬性（其值預設為空的陣列，打算使用內含 Employee 物件的陣列作為他的值）。
  • WorkerBee 也是 Employee 為基礎。他加入了 projects 屬性（其值預設為空的陣列，打算使用字串的陣列作為他的值）。
  • SalesPerson 以 WorkerBee 為基礎。他加入了 quota 屬性（其值預設為 100）。他也使用 "sales" 值覆寫了 dept 屬性，指明所有的銷售員都從屬於同一部門。
  • Engineer 以 WorkerBee 為基礎。他加入了 machine 屬性（其值預設為空字串），而且也使用 "engineering" 值覆寫了 dept 屬性。
