字串運算子

除了比較運算子可以用於字串值以外，連結運算子 (+) 把兩個字串值串連在一起，返回另一個以兩個運算元字串連結的字串。例如，"my " + "string" 返回 "my string" 字串。

代入運算子 += 的簡記法也可用於連結字串。例如，如果變數 mystring 內有 "alpha" 值，mystring += "bet" 表達式的求值為“alphabet”，並把這個值代入到 mystring。
