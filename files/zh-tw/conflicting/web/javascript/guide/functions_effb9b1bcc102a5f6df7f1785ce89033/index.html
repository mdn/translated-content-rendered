eval 函數

eval 函數無須參考實際的物件，就可以對 JavaScript 代碼的字串直接求值。eval 語法如下︰

eval(expr)

此處 expr 就是要被求值的字串。

如果字串表示成表達式，eval 就會對表達式求值。如果參數（expr）表示成一個以上的 JavaScript 語句，eval 就會對語句求值。eval 代碼的作用域等同於呼叫端代碼的作用域。不要用 eval 來對算術表達式求值，JavaScript 會自動的對算術表達式來求值。
