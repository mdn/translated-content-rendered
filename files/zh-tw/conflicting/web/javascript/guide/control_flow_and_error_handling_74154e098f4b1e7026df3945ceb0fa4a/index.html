註解

註解是作者用來解釋 Script 在做什麼的標記符號。註解會被解譯器忽略。JavaScript 支援 Java 和 C++ 形式的註解︰

  • 單行註解是在前面加上兩個斜線 (//)。
  • 多行註解是在前面加上 /* 並在尾端加上 */。

範例
以下範例示範這兩種註解︰

// 這是單行註解。

/* 這是多行註解。不管是什麼長度都可以，你也
可以在任何你喜歡的地方使用。 */
