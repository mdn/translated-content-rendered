JavaScript (JS) on kevyt, tulkattu, tai juuri-ajallaan knnetty ohjelmointikieli, joka pit funktioita "ensimmisen luokan kansalaisina". Vaikka se tunnetaan parhaiten verkkoselaimessa toimivana skriptauskielen, sit kyttvt mys monet ei-selainpohjaiset ympristt, kuten Node.js, Apache CouchDB ja Adobe Acrobat. JavaScript on prototyyppipohjainen, yksisikeinen, dynaaminen ohjelmointikieli, joka tukee monia eri ohjelmintitapoja, kuten olio-ohjelmointi-, imperatiivista sek funktionaalista ohjelmointityyli. Tm osa on omistettu pelkstn JavaScriptille ohjelmointikielen huomioimatta ominaisuuksia, jotka ovat eritysi verkkosivuille tai muille ajoympristille. Tarkempaa tietoa selainten ohjelmointirajapinnoista, joita kytetn verkkosivujen kehittmisess lytyy Web API- ja DOM-osioista.

JavaScriptin kyttm standardi on ECMAScript. Vuodesta 2012 lhtien kaikki modernit selaimet tukevat tysin ECMAScript 5.1:t ja sit vanhemmat selaimet tukevat vhintn ECMAScript 3:sta. 17. keskuuta 2015 ECMA International julkaisi kuudennen version ECMAScriptist, jonka virallinen nimitys on ECMAScript 2015, tt kutsuttiin alunperin ECMAScript 6 tai ES6:ksi. Siit lhtien, ECMAScript-standardia on pivitetty vuosittain. Tm dokumentaatio viittaa tmnhetkiseen vedokseen, mik on tll hetkell ECMAScript 2020.

JavaScripti ei kannata sekoittaa Java-ohjelmointikieleen. Molemmat "Java" ja "JavaScript" ovat Oraclen tuotemerkkej Yhdysvalloissa ja muissa maissa, mutta siit huolimatta nill kahdella ohjelmointikielell on hyvin erilainen syntaksi, semantiikka ja kyttkohteet.
