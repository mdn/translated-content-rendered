Cambios para desarrolladores

JavaScript

  • Los objetos Map y Set pasaron de tener un método size() a tener una propiedad size (error 807001)
  • Los objetos Map and Set ahora tienen un método clear(). (error 805003)

CSS

  • Support for the viewport-relative <length> units, vh, vw, vmin, and vmax, has landed (error 503720)
  • CSS Flexbox has been unprefixed, but remains disabled by default (error 801098).
  • The -moz-initial value has been unprefixed (error 806068). -moz-initial will be kept for a while as an alias; however, authors are strongly encouraged to switch over to initial.
  • The CSS text-transform property now supports the full-width keyword, which allows a more seamless inclusion of Latin characters in text using ideographic fixed-width characters, like Chinese or Japanese (error 774560).
  • The CSS page-break-inside has been implemented (error 685012).
  • The CSS calc() function can now be used on <color-stop> (on <gradient>).
  • The CSS @page at-rule is now supported (error 115199). Note that the pseudo-classes :first, :right, and :left are not yet implemented.
  • The :-moz-placeholder pseudo-class is replaced by the ::-moz-placeholder pseudo-element (error 737786).

DOM

  • The element.getElementsByTagName method will now return HTMLCollection (error 799464).
  • The File mozLastModifiedDate property has been implemented. (error 793955)
  • The CanvasRenderingContext2D isPointInStroke method has been implemented (error 803124).

XForms

Support for XForms has been removed in Firefox 19.

Changes for add-on and Mozilla developers

Note: A key change in Firefox 19 is that nsresult is now strongly typed. This will help make it easier to detect bugs that are caused by mishandling of return values, but may cause existing code to break if it's making incorrect assumptions in this regard.

  • getBrowserSelection() now returns the selected text in a text input field. As a result, gContextMenu.isTextSelected will be true when the user selects text in a text input field that is not a password field. (error 565717)
  • Dict.jsm: Dict() now takes a JSON String. Dict.toJSON() was added, and it returns a JSON String. (error 727967)
