Esta es una tecnología experimental
Comprueba la Tabla de compabilidad de navegadores cuidadosamente antes de usarla en producción.

La propiedad Window.URL devuelve un objeto que proporciona métodos estáticos usados para crear y gestionar objetos de URLs. Además puede ser llamado como un constructor para construir objetos URL.

Nota: Esta característica está disponible en Web Workers.

Sintaxis

Llamando a un método estático:

img.src = URL.createObjectURL(blob);

Construyendo un nuevo objeto:

var url = new URL("../cats/", "https://www.example.com/dogs/");
