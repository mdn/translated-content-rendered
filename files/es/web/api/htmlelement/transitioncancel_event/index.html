El evento transitioncancel es lanzado cuando una transición CSS es cancelada.

Véase GlobalEventHandlers.ontransitioncancel para mas información y ejemplos.

Información general

Interfaz
TransitionEvent
Burbuja
Sí

No
Objetivo
document, element

Ninguna

Propiedades

Propiedad Tipo Descripción
target Read only EventTarget El objetivo del evento (the topmost target in the DOM tree).
type Read only DOMString El tipo de evento.
bubbles Read only Boolean Si el evento normalmente se propaga o no
cancelable Read only Boolean Si el evento es cancelable o no
propertyNameRead only DOMString El nombre de la propiedad CSS asociada con la transición.
elapsedTimeRead only Float

La cantidad de tiempo que ha durado la transición, en segundos, desde el momento en que el evento fué generado. Este valor no se ve afectado por el valor de transition-delay.
pseudoElementRead only DOMString

El nombre (empezando con dos "dos puntos") del pseudo-elemento CSS en el que ha ocurrido la transición (en caso de que el objetivo del evento sea dicho pseudo-elemento correspondiente al elemento), o una cadena vacía si la transición ha ocurrido en un elemento (lo que quiere decir que el objetivo del evento es dicho elemento).
