Propiedades

Heredando propiedades de los elementos padres HTMLElement.

HTMLLabelElement.control Read only
Es un HTMLElement representando el control a la cual cada etiqueta esta asociado.
HTMLLabelElement.form Read only
Es un objeto HTMLFormElement representado el formulario al cual el control etiquetado está asociado, arrojando el valor null, si no hay asociado ningun control o si ese control no está asociado con un formulario. En otras palabras, esto es un atajo para: HTMLLabelElement.control.form.
HTMLLabelElement.htmlFor
Este es un contenedor de caracteres que contiene el ID del control etiquetado, esto es lo que refleja el atributo for

Métodos

No hay metodos específicos, hereda los metodos de los elelemtos padres, HTMLElement.
