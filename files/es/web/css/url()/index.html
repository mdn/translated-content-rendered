La función de CSS url() usa un filtro SVG para cambiar la apariencia en la imagen de entrada.

Sintaxis

url(location)

Parámetros

location
La <url> de un archivo XML que especifique un filtro SVG, y puede incluir un ancla a un elemento filtro específico.

Ejemplo

url(resources.svg#c1)
