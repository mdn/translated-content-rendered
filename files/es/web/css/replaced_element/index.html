Summary

Dentro de CSS tenemos los elementos de reemplazo, cuya representacion esta fuera del ambito de propio CSS. Son un tipo de objeto externo, por tanto su representacion es independiente de CSS. Algunos objetos que normalmente funcionan como objetos de reemplazo son <img>, <object>, <video> o elementos de formulario como <textarea>, <input>. Algunos elementos como <audio> or <canvas> ejercen como elementos de reemplazo solo en casos especificos. Los objetos insertados a traves de las propiedades CSS content son objetos de reemplazo anonimos..

CSS gestiona elementos de reemplazo en casos concretos, por ejemplo al calcular los margenes y algunos auto valores.

Recuerda que algunos elementos de reemplazo, no todos, tienen dimensiones intrinsecas o linea de base establecida, las cuales son utilizadas por propiedades de CSS como vertical-align.
