A continuación se muestra una lista alfabética de elementos de presentación en MathML.
La etiqueta de presentación es usada para describir la estructura y el diseño de la notación matemática mientras que la etiqueta de contenido proporciona el significado matemático subyacente y se supone que no se renderiza a través del parseador de MathML (ver error 276028). Si quieres aprender más sobre la etiqueta de contenido deberías echar un vistazo al Capítulo 4 en la especificación de MathML 3.

math

  • <math> (Elemento raíz)

A

  • <maction> (Acciones enlazadas a sub-expresiones)
  • <maligngroup> (Grupo de alineación) No implementado (mira error 122253)
  • <malignmark> (Puntos de alineación) No implementado (mira error 122253)

E

  • <menclose> (Enclosed contents)
  • <merror> (Enclosed syntax error messages)

F

  • <mfenced> (Paréntesis)
  • <mfrac> (Fracción)

G

  • <mglyph> (Mostrar símbolos no estándar) No implementado (mira error 297465)

I

  • <mi> (Identificador)

L

  • <mlabeledtr> (Fila etiquetada en una tabla o matriz) No implementado (mira error 689641)
  • <mlongdiv> (Notación de la división larga) No implementado (mira error 534967)

M

  • <mmultiscripts> (Prescripts and tensor indices)

N

  • <mn> (Número)

O

  • <mo> (Operador)
  • <mover> (Overscript)

P

  • <mpadded> (Espacio alrrededor del contenido)
  • <mphantom> (Contenido invisible con espacio reservado)

R

  • <mroot> (Radical con índice especificado)
  • <mrow> (Sub-expressiones agrupadas)

S

  • <ms> (String literal)
  • <mscarries> (Annotations such as carries) No implementado (mira error 534967)
  • <mscarry> (Single carry, child element of <mscarries>) No implementado (mira error 534967)
  • <msgroup> (Grouped rows of <mstack> and <mlongdiv> elements) No implementado (mira error 534967)
  • <msline> (Horizontal lines inside <mstack> elements) No implementado (mira error 534967)
  • <mspace> (Space)
  • <msqrt> (Square root without an index)
  • <msrow> (Rows in <mstack> elements) No implementado (mira error 534967)
  • <mstack> (Stacked alignment) No implementado (mira error 534967)
  • <mstyle> (Style change)
  • <msub> (Subscript)
  • <msup> (Superscript)
  • <msubsup> (Subscript-superscript pair)

T

  • <mtable> (Table or matrix)
  • <mtd> (Cell in a table or a matrix)
  • <mtext> (Text)
  • <mtr> (Row in a table or a matrix)

  • <munderover> (Underscript-overscript pair)
