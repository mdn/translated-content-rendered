Referencia de XSLT y XPath: Elementos XSLT, Funciones EXSLT, <ahref=" docs="" es="" funciones"="" xpath="">XPath:Funciones, XPath:Ejes </ahref=">


El elemento <xsl:fallback> indica la plantilla a usar en caso de que algun elemento extendido (o, eventualmente, una nueva versión) no esté soportado.

Sintaxis

<xsl:fallback>
	PLANTILLA
</xsl:fallback>

Atributos requeridos

Ninguno.

Atributos opcionales

Ninguno.

Tipo

Instrucción, aparece dentro de una plantilla.

Definido

XSLT, sección 15

Soporte en Gecko

Sin soporte por el momento.
