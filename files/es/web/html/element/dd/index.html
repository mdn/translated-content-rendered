El elemento HTML <dd> provee detalles acerca de o la definición de un término precedente (<dt>) en una lista de descripciones (<dl>).

La fuente para este ejemplo interactivo esta almacenado en un repositorio GitHub. Si desea contribuir al poryecto de ejemplos interactivos, por favor clone https://github.com/mdn/interactive-examples y envíe una petición de extracción (pull request).

Categorías de contenido Ninguna
Contenido permitido Contenido de flujo
Omisión de Tag El tag inicial es requerido. El tag final puede ser omitido si el elemento <dd> se encuentra inmediatamente seguido por otro elemento <dd>, o si no hay más contenido en el elemento padre.
Padres permitidos <dl> o (en WHATWG HTML) un <div> que se encuentra dentro de un <dl>.
Hermano previo <dt> u otro elemento <dd>.
Roles ARIA permitidos Ninguno
Interfaz DOM HTMLElement

Atributos

Los atributos de este elemento incluye los atributos globales.

nowrap
Si el valor de este atributo es definido como yes, el texto de la definición no se encapsulará. El valor por defecto es no.
