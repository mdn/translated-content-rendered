Los siguientes enlaces brindan soluciones puntuales a los problemas más comunes a los que te enfrentarás a diario en HTML.

Estructura básica

La principal aplicación de HTML es la estructuración del documento. Si eres nuevo en HTML debes empezar aquí.

  • Cómo crear un documento HTML básico
  • Cómo dividir una página web en secciones lógicas
  • Cómo establecer una estructura adecuada de encabezados y párrafos

Semántica básica textos

HTML se especializa en suministrar información semántica del documento, así que HTML soluciona muchas dudas que puede tener sobre como transmitir su mensaje a través de un documento.

  • Cómo crear una lista de elementos con HTML
  • Cómo enfatizar o acentuar contenido
  • Cómo indicar qué texto es importante
  • Cómo muestra el ordenador el codigo HTML
  • Cómo comentar imagenes y gráficas
  • Cómo marcar abreviaturas y hacerlas entendibles
  • Cómo añadir citas y referencias a sitios web
  • Cómo definir términos con HTML

Hipervínculos

Uno de las principales razones por las que HTML hace más fácil la navegación son los hipervínculos, que pueden ser usados de diferentes formas:

  • Cómo crear un hipervínculo
  • Cómo crear una tabla de contenidos con HTML

Imágenes y multimedia

  • Cómo añadir imagenes al sitio web
  • Cómo añadir contenidos de video al sitio web

Scripts y estilización

HTML únicamente define la estructura del documento. Para solucionar los problemas de presentación usamos CSS, o usamos scripts para hacer la página interactiva.

  • Cómo usar CSS dentro de un sitio web
  • Cómo usar JavaScript dentro de un sitio web

Contenido integrado

  • Cómo integrar una página web dentro de otra
  • Cómo integrar contenido Flash dentro de una página web

Problemas avanzados o raros

Más allá de lo basico, HTML es muy rico y ofrece características avanzadas para solucionar problemas complejos. Estos artículos te ayudarán a abordar estos casos menos comunes:

Formularios

Los formularios son una estructura compleja en HTML para enviar datos desde una pagina web al servidor. Te animamos a que revises la guia completa. Aquí es por donde deberías empezar:

  • Cómo crear un formulario Web simple
  • Cómo estructurar un formulario Web

Información tabular

Alguna información, llamada datos tabulares, necesita ser organizada en tablas mediante filas y columnas. Siendo ésta una de las estructuras más complejas de HTML, no es fácil dominarla:

  • Cómo crear una tabla de datos
  • Cómo hacer tablas HTML accesibles

HTML no es monolingüe. Éste proporciona herramientas para manejar problemas comunes de internacionalización.
