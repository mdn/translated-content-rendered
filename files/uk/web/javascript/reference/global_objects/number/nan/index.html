Властивість Number.NaN відображає "не число" (Not-A-Number). Є еквівалентом NaN.

Вам не потрібно створювати об'єкт Number, щоб звертатись до статичної властивості (використовуйте Number.NaN).

Атрибути поля Number.NaN
Доступний для запису ні
Доступний для переліку ні
Доступний для налаштування ні

Приклади

Перевірка на числове значення

function sanitise(x) {
  if (isNaN(x)) {
    return Number.NaN;
  }
  return x;
}
