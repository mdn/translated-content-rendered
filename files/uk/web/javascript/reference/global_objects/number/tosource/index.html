Obsolete
This feature is obsolete. Although it may still work in some browsers, its use is discouraged since it could be removed at any time. Try to avoid using it.

Метод toSource() вертає рядкове представлення першокоду об'єкта.

Синтаксис

numObj.toSource()
Number.toSource()

Значення, що вертається

Рядкове представлення першокоду об'єкта.

Приклади

Вбудована функція

Для вбудованого об'єкта Number, toSource() вертає наступний рядок, який вказує, що першокод недоступний:

function Number() {
    [native code]
}

Для екземплярів Number, toSource() вертає рядкове представлення першокоду.

Цей метод зазвичай викликається внутрішньо у JavaScript, а не явно у коді.
