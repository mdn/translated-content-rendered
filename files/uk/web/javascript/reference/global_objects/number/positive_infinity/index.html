Опис

Значення Number.POSITIVE_INFINITY є таким самим, як і значення властивості глобального об'єкта Infinity.

Це значення поводиться дещо інакше, ніж математична нескінченність:

  • Будь-яке позитивне значення, в тому числі POSITIVE_INFINITY, помножене на POSITIVE_INFINITY вертає POSITIVE_INFINITY.
  • Будь-яке від'ємне значення, в тому числі NEGATIVE_INFINITY, помножене на POSITIVE_INFINITY, вертає NEGATIVE_INFINITY.
  • Будь-яке позитивне число, розділене на POSITIVE_INFINITY, вертає нуль.
  • Будь-яке від'ємне число, розділене на POSITIVE_INFINITY, вертає від'ємний нуль.
  • Нуль, помножений на POSITIVE_INFINITY, вертає NaN.
  • Значення NaN, помножене на POSITIVE_INFINITY, вертає NaN.
  • Значення POSITIVE_INFINITY, розділене на будь-яке від'ємне значення, окрім NEGATIVE_INFINITY, вертає NEGATIVE_INFINITY.
  • Значення POSITIVE_INFINITY, розділене на будь-яке позитивне значення, окрім POSITIVE_INFINITY, вертає POSITIVE_INFINITY.
  • Значення POSITIVE_INFINITY, розділене на NEGATIVE_INFINITY чи POSITIVE_INFINITY, вертає NaN.

Ви можете скористатись властивістю Number.POSITIVE_INFINITY, щоб позначити помилку в умові, яка вертає скінченне число у випадку успіху. Зауважте, однак, що метод isFinite в цьому випадку більш доречний.

Оскільки POSITIVE_INFINITY є статичною властивістю об'єкта Number, її завжди слід використовувати як Number.POSITIVE_INFINITY, а не як властивість створеного вами об'єкта Number. 

Приклади

Використання POSITIVE_INFINITY

У наступному прикладі змінній bigNumber присвоюється значення, яке є більшим за максимальне. Коли виконується інструкція if, bigNumber має значення Infinity, а отже, bigNumber присвоюється більш кероване значення для продовження.

var bigNumber = Number.MAX_VALUE * 2;

if (bigNumber == Number.POSITIVE_INFINITY) {
  bigNumber = returnFinite();
}
