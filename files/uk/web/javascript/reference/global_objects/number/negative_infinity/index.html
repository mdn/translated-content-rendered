Опис

Значення Number.NEGATIVE_INFINITY є таким самим, як від'ємне значення властивості Infinity глобального об'єкта.

Це значення поводиться дещо інакше, ніж математична нескінченність:

  • Будь-яке позитивне значення, в тому числі POSITIVE_INFINITY (позитивна нескінченність), помножене на NEGATIVE_INFINITY (від'ємну нескінченність) дорівнює NEGATIVE_INFINITY.
  • Будь-яке від'ємне значення, в тому числі NEGATIVE_INFINITY, помножене на NEGATIVE_INFINITY, дорівнює POSITIVE_INFINITY.
  • Будь-яке позитивне значення, розділене на NEGATIVE_INFINITY, дорівнює від'ємному нулю.
  • Будь-яке від'ємне значення, розділене на NEGATIVE_INFINITY, дорівнює позитивному нулю.
  • Нуль, помножений на NEGATIVE_INFINITY, дорівнює NaN.
  • Значення NaN, помножене на NEGATIVE_INFINITY, дорівнює NaN.
  • Значення NEGATIVE_INFINITY, розділене на будь-яке від'ємне значення, за винятком NEGATIVE_INFINITY, дорівнює  POSITIVE_INFINITY.
  • Значення NEGATIVE_INFINITY, розділене на будь-яке позитивне значення, за винятком POSITIVE_INFINITY, дорівнює NEGATIVE_INFINITY.
  • Значення NEGATIVE_INFINITY, розділене на NEGATIVE_INFINITY чи POSITIVE_INFINITY, дорівнює NaN.

Ви можете скористатись властивістю Number.NEGATIVE_INFINITY, щоб позначити помилку в умові, яка повертає скінченне число у випадку успіху. Зауважте, однак, що метод isFinite в цьому випадку був би більш доречним.

Оскільки NEGATIVE_INFINITY є статичною властивістю Number, її завжди слід використовувати як Number.NEGATIVE_INFINITY, а не як властивість створеного вами об'єкта Number. 

Приклади

Використання NEGATIVE_INFINITY

У наступному прикладі змінній smallNumber присвоюється значення, яке є меншим за мінімальне. Коли виконується інструкція if, smallNumber має значення -Infinity, отже smallNumber присвоюється більш кероване значення для продовження.

var smallNumber = (-Number.MAX_VALUE) * 2;

if (smallNumber === Number.NEGATIVE_INFINITY) {
  smallNumber = returnFinite();
}
