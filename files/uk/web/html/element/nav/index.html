HTML-елемент <nav> (HTML елемент навігації) являє собою секцію сторінки, котра має посилання на інші сторінки або блоки в межах поточної сторінки.

Категорії контенту Потоковий контент, секційний контент, тактильний контент.
Дозволений контент Потоковий контент.
Пропуск закриваючого тега None, both the starting and ending tag are mandatory.
Дозволені батьківські елементи Будь-який елемент, що приймає потоковий контент.
DOM інтерфейс HTMLElement

Атрибути

Цей елемент може містити лише глобальні атрибути.

Зауваження щодо використання

  • Не всі посилання документа мають бути в елементі <nav>, бо він призначений для головних блоків навігаційних посилань. Наприклад, елемент <footer> зазвичай містить перелік посилань, які не потрібно розміщувати у елементі <nav>;
  • Документ може містити кілька елементів <nav> (наприклад, один для навігації по сайту, а інший — для навігції в межах сторінки);
  • Деякі програмні засоби для людей з вадами зору, часто можуть використовувати цей елемент щоб визначити, чи слід пропускати початкове відтворення даного вмісту.

Приклади

<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>
