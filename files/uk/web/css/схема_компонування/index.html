Схема компонування (layout mode) в CSS являє собою алгоритм, який визначає розташування та розміри елементів залежно від того, як вони взаємодіють із сусідніми та батьківськими елементами. Існує декілька таких схем:
  • Блокове компонування (block), призначене для розмітки документів. Ця схема має відповідні засоби, як-от можливість зробити елементи рухомими (плаваючими) або скомпонувати їх в кілька стовпчиків.
  • Рядкове компонування (inline), призначене насамперед для розмітки тексту.
  • Табличне компонування, призначене для розмітки таблицею.
  • Компонування розміщенням (positioned), призначене для застосування до елементів, що не надто взаємодіють з іншими.
  • Компонування flexbox, призначене для створення складної розмітки сторінок, які можна плавно масштабувати.
  • Компонування сіткою, призначене для розташування елементів відносно нерухомої сітки. 

Зауваження: Не всі властивості CSS застосовні до всіх схем компонування. Більшість із них вживаються разом з однією-двома схемами і жодним чином не діють, якщо відповідний елемент компонується за іншою схемою.
