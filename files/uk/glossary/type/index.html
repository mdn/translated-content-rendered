Тип (або тип даних) є характеристикою значення, що визначає формат його зберігання та способи обробки. Скажімо, ціле число можна помножити на дробове, але не на текстовий рядок.

Крім того, кожен тип даних передбачає обмежену форматом зберігання множину значень. Наприклад, тип Boolean у JavaScript може містити лише значення true та false, натомість тип String містить текстові рядки, а Number — цілі числа та десяткові дроби тощо.
