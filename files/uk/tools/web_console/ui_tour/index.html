Для відкриття Веб-консолі:

  • виберіть "Веб-консоль" з під-меню "Веб-розробка" меню Firefox (або з меню Інструменти, якщо активоване відображення панелі меню або в  Mac OS X)
  • або натисніть комбінацію клавіш CtrlShiftK (CommandOptionK в OS X)

В нижній частині вікна веб-переглядача з'явиться Панель інструментів з активованою веб-консоллю (на панелі інструментів DevTools вона просто називається "консоль"):

Інтерфейс Веб-консолі розділяється на три горизонтальні секції:

  • Панель інструментів: вздовж верхньої частини знаходиться панель інструментів, що містить дві кнопки. Натисніть кнопку "сміття", щоб очистити вміст консолі. Натисніть піктограму "воронка", щоб відфільтрувати повідомлення, які відображаються в консолі 
  • Командний рядок:  командний рядок починається з подвійних кутових дужок (>>). Використовуйте його для введення виразів JavaScript
  • Панель відображення повідомлення: тут відображаються повідомлення, що генеруються кодом на сторінці та команди, введені в командному рядку
