Монітор мережі надає дві області панелей інструментів: одну над основним розділом, а іншу під ним.

Панель інструментів

Панель інструментів знаходиться у верхній частині головного вікна Монітора мережі.

На ній знаходяться:

  • Піктограма для очищення списку мережних запитів.
  • Поле для вводу фільтру запитів за URL та за властивостями
  • Набір піктограм для фільтрування списку мережних запитів:
    • за типом змісту відповіді
    • XHR запити
    • оновлення WebSocket (позначені WS)
  • типово, Монітор мережі очищується кожного разу, коли Ви переходите до нової сторінки або перезавантажуєте поточну сторінку. Цю поведінку можна змінити, активувавши параметр "Постійне журналювання".
  • Параметр, який дозволяє вимкнути кешування
  • Обмеження швидкості передачі
  • HAR

Друга область панелі інструментів, що знаходиться нижче Монітора мережі, містить:

  • Піктограму запуску Аналізу швидкодії
  • Звітна інформація по кількості, загальному розміру і загальному часу запитів, що відображаються на цій сторінці .

Можливості Монітору мережі

У наступних статтях розглядаються різні аспекти використання Монітору мережі:

  • Список мережних запитів
  • Подробиці мережних запитів
  • Записування мережного трафіку
  • Аналіз швидкодії
  • Обмеження швидкості передачі
