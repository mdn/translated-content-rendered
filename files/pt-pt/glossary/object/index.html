Objeto refere-se a uma estrutura de dados contendo dados e instruções para trabalhar com os dados. Os objetos às vezes referem-se a coisas do mundo real, por exemplo, um carro ou objeto de mapa num jogo de carros. JavaScript, Java, C++, Python, e Ruby são exemplos de linguagens em programação orientada em objeto.
