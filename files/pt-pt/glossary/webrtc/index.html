WebRTC (Comunicações da Web em Tempo Real) é uma API que pode ser utilizada por aplicativos de videoconferência, chamadas de voz, e P2P.

O WebRTC consiste principalmente destas partes:

getUserMedia()
Concede acesso à câmera e / ou microfone de um dispositivo, e pode conectar os seus sinais a uma conexão RTC.
RTCPeerConnection
Uma interface para configurar videoconferências ou chamadas de voz.
RTCDataChannel
Fornece um método para estabelecer uma conexão P2P entre navegadores web.
