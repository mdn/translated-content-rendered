Uma ligação para um objeto. Referências de objeto podem ser utilziadas como objetos interligados.

O conceito de referências de objeto fica claro quando atribuir o mesmo objeto a mais de uma propriedade. Em vez de manter uma cópia do objeto, cada propriedade atribuída contém referências de objeto que se vinculam ao mesmo objeto, de modo que, quando o objeto altera todas as propriedades referentes ao objeto, refletem a alteração.
