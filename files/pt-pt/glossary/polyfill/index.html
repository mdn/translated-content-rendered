Um polyfill é um pedaço de código (geralmente JavaScript) usado para fornecer funcionalidades modernas em browsers mais antigos que não o suportam nativamente.

Por exemplo, um polyfill pode ser usado para simular a funcionalidade de um elemento HTML Canvas no Microsoft Internet Explorer 7, usando o plugin Silverlight, ou simular o suporte para unidades rem em CSS, ou text-shadow, ou qualquer outra coisa que você queira.
