Em JavaScript, BigInt é um tipo de dados numéricos que pode representar inteiros num formato de precisão arbitrária. Em outras linguagens de programação podem existir diferentes tipos de dados numéricos, por exemplo: Inteiros, Flutuadores, Duplos, ou Bignums.
