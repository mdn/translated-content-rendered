O elemento video é usado para inserir conteúdo de vídeo em um documento HTML ou XHTML. O elemento video foi adicionado como parte do HTML 5.

  • Tipo de Elemento: block-level
  • Conteúdo Permitido: alternate text
Nota: Atualmente, o Firefox suporta apenas Theora para vídeo, em recipientes Ogg.  Além disso, o servidor precisa servir o arquivo utilizando o MIME type correto para que o Firefox o reproduza corretamente.

Atributos

Note que diversos atributos ainda não são suportados pelo Firefox.

autoplay
Um atributo booleano; se especificado, o vídeo iniciará a reprodução automaticamente assim que puder, portanto, sem para para finalizar o carregamento dos dados.
autobuffer
Um atributo booleano; se especificado, o vídeo iniciará a leitura dos dados (buffering) mesmo se não estiver configurado para reproduzir automaticamente. Isto deve ser usado para casos em que provavelmente o vídeo será reproduzido (por exemplo, se o usuário navegou para a página exclusivamente para reproduzir o vídeo, não se o vídeo foi inserido junto a outro conteúdo). O vídeo é lido (buffered) até que o cache de mídia esteja cheio.
controls
Se este atributo estiver presente, Firefox oferecerá controles para permitir que o usuário controle a reprodução do vídeo, incluindo volume, procura, e pausa/continuar.
height
A altura da área de exibição do vídeo, em pixels CSS.
loop Unimplemented bug 449157
Um atributo booleano; se especificado, ao alcançar o fim do vídeo, voltará ao início automaticamente.
poster Unimplemented bug 449156
Uma URL indicando um quadro pôster para se exibir enquanto não houver dados do vídeo disponíveis. Se este atributo não for especificado, nada é exibido até o vídeo estar disponível para reprodução.
src
A URL do vídeo a ser embutido. Este atributo é opcional; em vez disso você pode usar o elemento source (en) dentro do bloco video para especificar o vídeo a embutir.
width
A largura da área de exibição do vídeo, em pixels CSS.

O deslocamento de tempo é especificado como um valor de ponto flutuante indicando o número de segundos do deslocamento.

Nota: A definição do valor de deslocamento de tempo ainda não está completa na especificação do HTML 5 e está sujeita a mudanças.

Exemplos

<video src="videofile.ogg" autoplay>
  Seu navegador não suporta o elemento <code>video</code>.
</video>

Reproduz um vídeo, inicia a reprodução assim que tenha recebido o bastante do vídeo para permitir a reprodução sem pausas para baixar mais.
