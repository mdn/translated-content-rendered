De Window.console alleen-lezen-eigenschap geeft een referentie terug aan het Console-object, die methodes voorziet om informatie te loggen naar de console van de browser. Deze methodes zijn enkel voorzien voor debuggingdoeleinden en zouden niet mogen worden gebruikt om informatie aan eindgebruikers te presenteren.

Syntaxis

var consoleObj = window.console;

Voorbeelden

Naar de console loggen

Dit eerste voorbeeld logt tekst naar de console.

console.log("Er is een fout ontstaan tijdens het laden van de inhoud");

Dit volgend voorbeeld logt een object naar de console. De velden van het object zijn uitbreidbaar met behulp van disclosure-widgets.

console.dir(someObject);

Zie Usage in [Page not yet written] voor uitgebreide voorbeelden.

Specificaties

Specificatie Status Commentaar
Console API Living Standard Aanvankelijke definitie.
