序列化（Serialization ）意味着将 对象 或某种其他类型的数据结构转换为可存储格式（例如，文件或 buffer）。

在 JavaScript 中，你可以通过调用 JSON.stringify() 函数将某个值序列化为 JSON 格式的 字符串。

CSS 值可以通过调用 CSSStyleDeclaration.getPropertyValue() 函数来序列化。
