    在网页中查找文本，并突出显示匹配项。

    要使用此API，您需要具有“查找” 权限。

    功能

    find.find()
    在网页中查找文本。
    find.highlightResults()
    突出显示找到的最后一组匹配项。
    find.removeHighlighting()
    删除所有突出显示。
