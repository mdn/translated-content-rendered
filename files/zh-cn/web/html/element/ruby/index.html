HTML <ruby> 元素 被用来展示东亚文字注音或字符注释。

内容范畴 流式内容, 短语内容, palpable content.
允许内容 短语内容
标签省略 不允许，开始标签和结束标签都不能省略。
允许的父级元素 See prose
DOM 接口 HTMLElement

属性

本元素支持 全局属性。

示例

示例1: 字

<ruby>
  汉 <rp>(</rp><rt>han</rt><rp>)</rp>
  字 <rp>(</rp><rt>zi</rt><rp>)</rp>
</ruby>

示例2: 词

<ruby>
  明 日 <rp>(</rp><rt>ming ri</rt><rp>)</rp>
</ruby>
