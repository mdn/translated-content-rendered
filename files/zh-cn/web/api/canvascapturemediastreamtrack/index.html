CanvasCaptureMediaStream 接口表示 MediaStream 对 HTMLCanvasElement 元素进行实时画面捕捉的内容。

属性

此接口继承了其父类 MediaStream 与 EventTarget 的属性。

CanvasCaptureMediaStream.canvas 只读
可返回当前媒体流所对应的 HTMLCanvasElement 元素对象。

方法

此接口继承了其父类 MediaStream 与 EventTarget 的方法。

CanvasCaptureMediaStream.requestFrame()
手动获取媒体流的一帧。 可以捕捉部分渲染帧画面.
