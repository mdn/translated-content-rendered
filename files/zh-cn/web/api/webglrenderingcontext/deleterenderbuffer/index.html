WebGL API 的 WebGLRenderingContext.deleteRenderbuffer() 方法用来删除给定的 WebGLRenderbuffer 对象. 如果渲染缓冲区已被删除，则此方法无效.

语法

void gl.deleteRenderbuffer(renderbuffer);

参数

renderbuffer
将要删除的{domxref("WebGLRenderbuffer")}} 对象.

返回值

None.

示例

删除一个渲染缓冲区

var canvas = document.getElementById('canvas');
var gl = canvas.getContext('webgl');
var renderbuffer = gl.createRenderbuffer();

// ...

gl.deleteRenderbuffer(renderbuffer);

说明

Specification Status Comment
WebGL 1.0
deleteRenderbuffer		 Recommendation 初始定义.
OpenGL ES 2.0
glDeleteRenderbuffers		 Standard OpenGL API手册(类似).
