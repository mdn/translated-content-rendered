非标准
该特性是非标准的，请尽量不要在生产环境中使用它！

此事件是早期版本的规范中的一个提案。不要依赖它。

beforescriptexecute 事件在一个脚本被执行前触发，取消此事件可以阻止该脚本的执行。

这是一个 Gecko（Firefox）特有的私有事件。
