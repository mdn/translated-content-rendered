这是一个实验中的功能
此功能某些浏览器尚在开发中，请参考浏览器兼容性表格以得到在不同浏览器中适合使用的前缀。由于该功能对应的标准文档可能被重新修订，所以在未来版本的浏览器中该功能的语法和行为可能随之改变。

 

设备加速对象可以提供有关设备沿三个轴（x、y、z）的加速度的信息。

属性

DeviceAcceleration.x 只读
沿X轴的加速度量（只读）
DeviceAcceleration.y 只读
沿Y轴的加速度量（只读）
DeviceAcceleration.z 只读
沿Z轴的加速度量（只读）
