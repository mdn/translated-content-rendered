当一个资源的加载已中止时，将触发 abort事件。

译者注：这个事件只在 IE 支持，试了最新的 Chrome、FireFox、Safari 都不支持。

常规信息

规范
DOM L3
接口
从UI组件产生为UIEvent , 否则为Event .
是否冒泡
否
可取消默认行为
否
目标对象
元素（Element）
默认行为
无

属性

属性 类型 描述
target 只读 EventTarget 产生该事件的对象(DOM树中最顶级的那个对象).
type 只读 DOMString 事件类型.
bubbles 只读 Boolean 该事件是否冒泡.
cancelable 只读 Boolean 该事件是否可取消默认行为.
view 只读 WindowProxy document.defaultView (该文档的window对象 )
detail 只读 long (float) 0.
