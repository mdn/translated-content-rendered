  1. Slotable

Slotable mixin 定义了允许节点成为 <slot> 元素的内容的特性——下面的特性被包含在 Element 和 Text API 之中。

属性

Slotable.assignedSlot 只读
返回节点所插入的 <slot> 元素。

无。
