Document Object Model הינו API עבור מסמכי HTML ו-XML. הוא מספק ייצוג מבני של המסמך המאפשר הן את שינוי תוכן המסמך ואת מראהו. עיקר תפקידו של ממשק זה, לאפשר לנו לחבר בין דפי אינטרנט לבין שפות תכנות.
