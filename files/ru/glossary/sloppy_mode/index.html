ECMAScript 5 и более поздние версии выполняют скрипты в Строгом режиме, который определенным образом изменяет семантику JavaScript для улучшения стабильности, и который упрощает понимание того, что происходит в программе при возникновении проблем.

Обычный, нестрогий, режим выполнения JavaScript иногда называется Грязным режимом (Sloppy mode). Это неофициальное название, но вы, вероятно, сталкивались с ним при серьёзной разработке на JavaScript.
