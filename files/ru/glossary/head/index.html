--- title: Head (Заголовок) slug: Словарь/Заголовок translation_of: Glossary/Head ---

Head (заголовок) это часть HTML документа, которая содержит metadata о документе, такие как автор, описание, ссылки на CSS или JavaScript файлы, которые должны быть применены в HTML.
