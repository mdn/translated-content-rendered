В JavaScript блок это набор связанных statements заключенных в скобки ("{}"). Например, можно поместить блок инструкций после блока if (condition), чтобы интерпретатор выполнял код в блоке, если условие имеет значение true, или пропускал целый блок, если значение условия false.
