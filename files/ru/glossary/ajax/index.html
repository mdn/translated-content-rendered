Асинхронный JavaScript And XML (AJAX) - это практика программирования для создания более сложных, динамических веб-страниц с использованием технологии, известной как XMLHttpRequest..

AJAX позволяет вам просто обновлять части DOM HTML веб-страницы вместо того, чтобы заново загружать всю страницу целиком. AJAX также позволяет вам работать асинхронно, что означает, что ваш код продолжает работать, пока обновляемая часть вашей веб-страницы пытается перезагрузиться (по сравнению с синхронным, который блокирует ваш код до тех пор, пока часть вашей веб-страницы не будет перезагружена полностью).

Благодаря интерактивным веб-сайтам и современным веб-стандартам, AJAX постепенно заменяется функциями в рамках JavaScript и официальным Fetch API стандартом.
