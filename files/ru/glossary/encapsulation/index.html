Инкапсуляция - это упаковка данных и functions в один компонент (например, class) и последующий контроль доступа к этому компоненту, создавая тем самым "черный ящик" из object. По этой причине, пользователю необходмо знать только интерфейс этого класса (то есть данные и функции, предоставляемые для взаимодействия с классом извне), а не то, как он реализован внутри.
