Для подключенного к сети компьютера с IP address, порт – это конечная точка входа для коммуникаций. Порты обозначаются числами, и до 1024 каждому порту по умолчанию назначен некоторый протокол.

Например, порт по умолчанию для протокола HTTP – 80, а порт для протокола HTTPS – 443, таким образом HTTP-сервер ожидает входящие запросы на данных портах. Каждый интернет протокол имеет порт по умолчанию: SMTP (25), POP3 (110), IMAP (143), IRC (194) и так далее.
