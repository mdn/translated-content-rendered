В JavaScript, истинное значение - это значение, которое считается true, когда встречается в контексте Boolean. Все значения истинные, если они не определены как falsy (кроме false, 0, -0, 0n, "", null, undefined, и NaN).

JavaScript использует приведение типов в Boolean контекстах.

Примеры истинных значений в JavaScript (которые будут приведены к true в булевых выражениях, и таким образом выполниться блок if):

if (true)
if ({})
if ([])
if (42)
if ("0")
if (new Date())
if (-42)
if (12n)
if (3.14)
if (-3.14)
if (Infinity)
if (-Infinity)
