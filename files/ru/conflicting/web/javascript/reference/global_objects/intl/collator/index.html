Сводка

Свойство Intl.Collator.prototype представляет объект прототипа конструктора Intl.Collator.

Атрибуты свойстваIntl.Collator.prototype
Записываемое нет
Перечисляемое нет
Настраиваемое нет

Описание

Смотрите Collator для описания экземпляров Intl.NumberFormat.

Экземпляры Intl.Collator наследуются от Intl.NumberFormat.prototype. Изменения объекта прототипа наследуются всеми экземплярами Intl.Collator.

Свойства

Ссылка на Intl.Collator.
Intl.Collator.prototype.compare
Геттер; возвращает функцию, сравнивающую две строки согласно порядку сортировки этого объекта Collator.

Методы

Intl.Collator.prototype.resolvedOptions()
Возвращает новый объект со свойствами, отражающими локаль и опции сравнения строк, вычисленные при инициализации объекта.
