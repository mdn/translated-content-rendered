url() - это CSS функция, использующая SVG filter для измения внешнего вида у выводимого изображения.

Синтаксис

url(расположение)

Параметры

расположение
В <url-адрес> из XML указывается файл, который задает фильтр SVG, а также может включать в себя привязки к конкретному фильтрующему элементу.

Пример

url(resources.svg#c1)
