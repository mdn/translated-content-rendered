Простой заголовок (или заголовок запроса с поддержкой безопасности CORS) - это один из следующих HTTP заголовков:

  • Accept,
  • Accept-Language,
  • Content-Language,
  • Content-Type с MIME-типом, найденым в этом значении (исключая параметры), либо application/x-www-form-urlencoded, multipart/form-data или text/plain.

Или один из этих клиентских заголовков:

  • DPR
  • Downlink
  • Save-Data
  • Viewport-Width
  • Width

Если они содержат только простые заголовки, запросы считаются простыми и не нужно отправлять preflight request в контексте CORS.
