Событие "abort" вызывается когда загрузка какого-либо ресурса была прервана.

Общая информация

Спецификация
DOM L3
Интерфейс
UIEvent если событие сгенерировано из пользовательского интерфейса, иначе Event.
Всплывание
Нет
Отменяемость
Нет
Цель
window, Element
Действие по умолчанию
Нет

Свойства

Свойство Тип Описание
target Только для чтения EventTarget Цель события (самый вышележащий элемент в DOM дереве).
type Только для чтения DOMString Тип события.
bubbles Только для чтения Boolean Поднимается ли событие вверх как принято или нет.
cancelable Только для чтения Boolean Является ли событие отменяемым или нет?
view Только для чтения WindowProxy document.defaultView (свойство window  объекта document)
detail Только для чтения long (float) 0.
