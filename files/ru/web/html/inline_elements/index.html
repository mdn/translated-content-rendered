Определение

В HTML (Языке Гипертекстовой Разметки), строчные элементы это те, которые занимают только то пространство, которое ограничено тегами, определяющими строчный элемент (необходимое для отображения их содержимого) и не нарушая потока содержимого (не требующее новой строки после каждого элемента). В этой статье мы рассмотрим встроенные элементы HTML и то, как они отличаются от  блочных элементов.

Строчный элемент не начинается с новой строки и в ширину занимает столько места, сколько это необходимо для его отображения.

Пример строчного элемента

HTML

<p>Этот элемент <span>span</span> строчный; цвет его фона изменен, чтобы показать начало и конец элемента.</p>

CSS

span { background-color: #8ABB55; }

Строчные vs. блочные

Модель контента
Как правило, встроенные элементы могут содержать только данные и другие встроенные элементы. Вы не можете помещать элементы блока внутри встроенных элементов.
Форматирование
По умолчанию встроенные элементы не заставляют новую строку начинаться в потоке документа. С другой стороны, элементы блока обычно вызывают разрыв строки (хотя, как обычно, это можно изменить с помощью CSS).

Элементы

Следующие элементы являются строчными:

  • b, big, i, small, tt
  • abbr, acronym, cite, code, dfn, em, kbd, strong, samp, time, var
  • a, bdo, br, img, map, object, q, script, span, sub, sup
  • button, input, label, select, textarea
