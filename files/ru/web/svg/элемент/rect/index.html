<rect>  - это базовая SVG фигура, используется для отрисовки прямоугольников по координатам угла, длины и высоты прямоугольника. Также может использоваться для отрисовки прямоугольников со скругленными углами.

Контекст использования

Категория Простая фигура, Графический элемент
Разрешенное содержимое Любое количество указанных элементов в любом порядке:
Элементы анимации »
Элементы описания »
Нормативный документ SVG 1.1 (2ое издание)

Пример

Простой пример использования rect

 

<?xml version="1.0"?>
<svg width="120" height="120"
     viewBox="0 0 120 120"
     xmlns="http://www.w3.org/2000/svg">

  <rect x="10" y="10" width="100" height="100"/>
</svg>

 

» rect-1.svg

<rect> со скругленными углами

 

<?xml version="1.0"?>
<svg width="120" height="120"
     viewBox="0 0 120 120"
     xmlns="http://www.w3.org/2000/svg">

  <rect x="10" y="10"
        width="100" height="100"
        rx="15" ry="15"/>

</svg>
