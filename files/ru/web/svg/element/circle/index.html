Элемент circle — базовая SVG фигура, используется для создания кругов c помощью координат центра и размера радиуса.

Информация

КатегорииБазовые фигуры, Графические элементы, Фигура
Разрешённый контентЛюбое количество следующих элементов, в любом порядке:
Элементы анимации
Элементы описания

Пример

<?xml version="1.0"?>
<svg viewBox="0 0 120 120" version="1.1"
  xmlns="http://www.w3.org/2000/svg">
  <circle cx="60" cy="60" r="50"/>
</svg>
