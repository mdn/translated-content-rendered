Свойство SVGAElement.target для чтения только SVGAElement возвращает объект SVGAnimatedString, который указывает часть целевого окна, фрейма, панель, в которую открывается при активации ссылки.

Это свойство используется, когда существует множество возможных целей для конечного ресурса, например, когда родительский документ является документом HTML или HTML-документом mlti-frame.

 

Синтаксис

myLink.target = 'value';

Стоимость

SVGAnimatedString, указывающий конечную цель ресурса, которая открывает документ при активации ссылки.
