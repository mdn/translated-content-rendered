  1. Web

Событие readystatechange срабатывает, когда изменяется атрибут документа readyState.

Основная информация

Спецификация
HTML5
 
Интерфейс
Event
Всплывает
Нет
Отменяемое
Нет
Цель
Document
Действие по умолчаанию
Нет

Свойства

Свойство Тип Описание
target Только для чтения EventTarget Цель события (Самая верхняя цель в дереве DOM).
type Только для чтения DOMString Тип события.
bubbles Только для чтения Boolean Всплывает ли событие.
cancelable Только для чтения Boolean Возможно ли отменить событие.

Примеры

document.readyState === "complete";
// true


// Альтернатива DOMContentLoaded
document.onreadystatechange = function () {
    if (document.readyState === "interactive") {
        initApplication();
    }
}

Поддержка браузерами

Данное событие давно поддерживается Internet Explorer и может быть использовано в качестве альтернативы событию DOMContentLoaded (см. примечание [2] в разделе  Поддержка браузерами).
