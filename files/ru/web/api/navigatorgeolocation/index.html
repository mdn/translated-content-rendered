NavigatorGeolocation содержит метод, позволяющий объектам реализовывать его,, получая Geolocation экземпляр объекта.

Здесь нет объектов типа NavigatorGeolocation, но некоторые интерфейсы, например, Navigator реализуют его.

Свойства

Интерфейс NavigatorGeolocation не наследует каких-либо свойств.

NavigatorGeolocation.geolocation Только для чтения
Возвращает объект Geolocation позволяющий получить доступ к местоположению устройства.

Методы

Интерфейс NavigatorGeolocation ни реализует, ни наследует  никаких методов.
