  1. Slotable

Миксин Slotable определяет возможности, которые позволяют нодам становиться контентом элемента <slot> — следующие возможности включены в  Element и Text.

Свойства

Slotable.assignedSlot Только для чтения
Возвращает <slot>, в который вставлена нода.
