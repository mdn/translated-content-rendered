Beschreibung

Siehe im Beitrag Collator für eine Beschreibung von Intl.Collator Instanzen.

Intl.Collator Instanzen erben von Intl.Collator.prototype. Änderungen am Prototypobjekt werden an alle Intl.Collator Instanzen vererbt.
