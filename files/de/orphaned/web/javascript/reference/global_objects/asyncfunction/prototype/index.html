Die AsyncFunction.prototype Eigenschaft repräsentiert den Prototypen des AsyncFunction Objektes.

Beschreibung

AsyncFunction Objekte erben von AsyncFunction.prototype. AsyncFunction.prototype kann nicht verändert werden.

Eigenschaften

AsyncFunction.constructor
Der Initialwert ist AsyncFunction.
AsyncFunction.prototype[@@toStringTag]
Gibt "AsyncFunction" zurück.
