Das <head> HTML Element legt den Kopf eines Dokuments fest. In diesem werden Informationen fÃ¼r das Dokument angegeben, darunter der Dokumententitel und Meta-Daten.

Beispiel

<html>
  <head>
    <title>Seitentitel</title>
  </head>
</html>
