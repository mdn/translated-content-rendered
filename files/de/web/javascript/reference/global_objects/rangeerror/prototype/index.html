Die RangeError.prototype Eigenschaft repräsentiert den Prototypen des RangeError Konstruktoren.

Eigenschaftsattribute von RangeError.prototype
Schreibbar nein
Aufzählbar nein
Konfigurierbar nein

Beschreibung

Alle RangeError Instanzen erben vom RangeError.prototype. Man kann den Prototypen nutzen um Eigenschaften oder Methoden allen Instanzen hinzuzufügen.

Eigenschaften

RangeError.prototype.constructor
Spezifiziert die Funktion welche den Prototypen einer Instanz erschaffen hat.
RangeError.prototype.message
Fehlermeldung. Obwohl ECMA-262 angibt, dass RangeError seine eigene message Eigenschaft versorgen sollte, erbt es in SpiderMonkey die Error.prototype.message.
RangeError.prototype.name
Fehlername. Geerbt von Error.
RangeError.prototype.fileName
Der Dateipfad verursacht diesen Fehler. Geerbt von Error.
RangeError.prototype.lineNumber
Die Zeile in der Datei, bei der der Fehler auftritt. Geerbt von Error.
RangeError.prototype.columnNumber
Die Spaltennummer der Zeile , bei der der Fehler auftritt. Geerbt von Error.
RangeError.prototype.stack
Stack - Ablaufverfolgung. Geerbt von Error.

Methoden

 Obwohl das RangeError Prototypobjekt keine eigenen Methoden beinhaltet, beerben RangeError Instanzen einige Methoden durch die Prototypenkette.
