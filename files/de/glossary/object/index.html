Ein Objekt bezieht sich auf eine Datenstruktur welche Daten und Anweisungen beinhaltet. Objekte stellen manchmal Gegenstände aus der echten Welt dar, wie zum Beispiel ein Auto oder eine Karte in einem Rennspiel. JavaScript, Java, C++, Python und Ruby sind Beispiele für  Objektorientierte Programmiersprachen.
